## Section4
Gitのコマンド

よく使われるGitコマンド：
- git init：新しいリポジトリを初期化
- git clone：既存のリポジトリをクローン
- git add：ステージングエリアにファイルを追加
- git commit：変更をコミット
- git push：ローカルリポジトリをリモートにプッシュ
- git pull：リモートリポジトリからプル
- git branch：ブランチを作成・削除・一覧表示
- git merge：ブランチをマージ
- git log：コミット履歴を表示
